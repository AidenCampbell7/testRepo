Customer Segmentation Explorer

This interactive tool segments customers into clusters based on metrics such as age and annual revenue. You can choose the number of clusters and see a scatter plot of customers coloured by their cluster. The project demonstrates an implementation of the K‑means algorithm in JavaScript and highlights how unsupervised learning can reveal meaningful groups for targeted marketing and sales strategies.
